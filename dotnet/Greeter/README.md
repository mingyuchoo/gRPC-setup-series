# Greet with F#

Grpc example written in F# language by freebsdk

## Prerequisites

### For macOS

```bash
$ brew install protobuf
```

### For Ubuntu Linux

```bash
$ sudo apt update
$ sudo apt install -y protobuf-compiler
```

### For Windows 11

- `https://github.com/protocolbuffers/protobuf/releases`
